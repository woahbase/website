---
Run
---

Run `weechat` with,

``` sh
docker run --rm -it \
  --name docker_weechat \
  --workdir /home/alpine \
  -p 9001:9001 \
  -v $PWD/data:/home/alpine/.weechat \
woahbase/alpine-weechat \
  weechat
```
