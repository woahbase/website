Run
---

Running the container starts the service.

``` sh
docker run --rm -it \
  --name docker_libresonic \
  -p 4040:4040 \
  -v $PWD/config:/config \
  -v $PWD/music:/music \
  -v $PWD/playlists:/playlists \
woahbase/alpine-libresonic:x86_64
```
