Run
---

Running the container starts the host monitor service.

``` sh
docker run --rm -it \
  --cap-add SYS_PTRACE \
  --name docker_netdata \
  -p 19999:19999 \
  -v /etc/hosts:/etc/hosts:ro \
  -v /etc/localtime:/etc/localtime:ro \
  -v /proc:/host/proc:ro \
  -v /sys:/host/sys:ro \
  -v $PWD/netdata:/etc/netdata \
woahbase/alpine-netdata:x86_64
```
